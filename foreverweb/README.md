# ForeverWeb — Starter Shell (scaffold)

This is a minimal starter scaffold for the ForeverWeb shell: Vite + React + Tailwind (TypeScript).

Quick start

Install dependencies and run the dev server:

```bash
cd foreverweb
npm install
npm run dev
```

Files created
- index.html — app entry
- src/* — React app
- tailwind.config.cjs, postcss.config.cjs
- vite.config.ts, tsconfig.json
- manifest.template.json — app manifest template
- scripts/fw — mock CLI
